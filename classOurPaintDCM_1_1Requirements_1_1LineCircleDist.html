Detailed Description

Constraint: Distance between a line and a circle.

This requirement enforces that the distance between a line and a circle equals a given fixed value. The distance is measured from the closest point on the line to the circle's center, minus the circle's radius.

Convert the constraint to a function for solving.

The distance between line L \((Ax + By + C = 0)\) and circle with center \((x_c, y_c)\)and radius \(r\) is:

\[ F = \frac{|A x_c + B y_c + C|}{\sqrt{A^2 + B^2}} - r - d = 0 \]

where \(d\) is the desired fixed distance.
