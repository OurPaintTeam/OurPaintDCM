Detailed Description

Constraint: Fixed distance between two points.

This requirement enforces a specific Euclidean distance between two points in 2D space. The constraint maintains the exact distance value regardless of the points' positions.

Convert the constraint to a function for solving.

Creates a function tree representing the constraint equation:

\[ (x_2 - x_1)^2 + (y_2 - y_1)^2 - d^2 = 0 \]

where \(d\) is the required distance.
