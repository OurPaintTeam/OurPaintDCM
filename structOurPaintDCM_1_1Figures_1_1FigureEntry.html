Detailed Description

Structure to store metadata about a geometric object.

Stores:

  • type: figure type (Point, Line, Circle, Arc)
  • ptr: raw pointer to object (void* for universality)

Why void*? This allows storing all types in a single unordered_map. When retrieving, we know the type from type field and perform safe static_cast.
