Detailed Description

Abstract base class for all geometric constraint functions.

Each requirement function represents a mathematical constraint f(x), used in geometric solvers or optimization systems.

The function typically computes:

  • evaluate(): deviation from the desired condition
  • gradient(): partial derivatives with respect to all variables
