Detailed Description

template<std::size_t N>
struct OurPaintDCM::Figures::PointBase< N >

Generic base class for N-dimensional points.

This template represents a point in N-dimensional space. coordinats is stored as a double value. And this is serves as a base for specialized classes like Point2D, Point3D etc.

Template Parameters
NNumber of dimensions. Must be greater than zero.
