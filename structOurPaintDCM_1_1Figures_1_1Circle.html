Detailed Description

template<typename PointT>
struct OurPaintDCM::Figures::Circle< PointT >

Generic circle structure for 2D/3D points.

Circle stores a pointer to the center point so that any modification of the point coordinates is automatically reflected in the circle.

Template Parameters
PointTType of point (e.g., Point2D, Point3D) derived from PointBase<N>.
