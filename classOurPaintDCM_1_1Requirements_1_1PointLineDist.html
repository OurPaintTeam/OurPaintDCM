Detailed Description

Requirement: fixed distance between point and sections.

This requirement enforces that the Euclidean distance between point and section is equal to the specified value.

Build the function for the point section distance constraint.

The equation is derived from the general line equation through segment endpoints A(xₐ, yₐ), B(xᵦ, yᵦ):

\[ \begin{aligned} Ax + By + C &= 0 \\ A &= y_B - y_A \\ B &= x_A - x_B \\ C &= x_B y_A - x_A y_B \end{aligned} \]

For a point P(x, y), the signed distance to line AB is:

\[ F = \frac{A x_P + B y_P + C}{\sqrt{A^2 + B^2}} \]

The requirement enforces:

\[ F - d = 0 \]

where \( d \) is the fixed distance value.
